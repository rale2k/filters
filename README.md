# filters
An application which has a button to create new filter and representation of already created filters.

## Frontend
React 19 + Vite + MaterialUI
NodeJS 22 recommended

```sh 
npm i 
npm run dev ->
```

## Backend
Spring Boot 3.5.2 + H2 DB + Flyway
Java 21 recommended

```sh 
chmod +x ./gradlew
./gradlew bootRun
```
